QUESTÕES SOBRE PROTOTIPAÇÃO DE INTERFACES
Aluno: Bruno Souza Montanari Data: 13/05/2024
1. O que é o Balsamiq Wireframes? Qual estilo de prototipação ele utiliza?

É um software para prototipação de interfaces que tem como objetivo ser rápido e ter o foco no conteúdo, sem se preocupar com os detalhes mínimos como cores e animações.

2. O que é o figma? Qual estilo de prototipação ele utiliza?

Esta também é uma ferramenta usada para prototipação de interfaces mas que tem mais opçôes de customização e mais detalhes para serem administrados.

3. Comente sobre um problema da sociedade que possa ser resolvido por um sistema computacional e porque resolver tal problema é importante.

Um problema que pode ser resolvido é o controle de desperdício de alimentos que poderia mostrar notificações dos alimentos que irão passar da validade, exibir dados de quantos Kgs de alimento vão para o lixo e muito mais. Esse sistema seria importante para evitar gastos desnecessários comprando muita comida e também para reduzir a taxa de desperdicio desses alimentos.
